# Brand Visual Identity Workflow

This workflow focuses on establishing a consistent visual style for a brand or project.

1. **Brand Discovery (CMO)**
   - Call `agent-media-cmo` to define the brand's core values, personality, and tone.

2. **Style Concept & Palette (Designer)**
   - Call `agent-media-designer` to generate a mood board (multiple image variations) and define a harmonious color palette.

3. **Asset Iteration**
   - Provide feedback to the Designer until the core logo concepts and brand patterns are finalized.

4. **Style Guide Generation (Copywriter)**
   - Call `agent-media-copywriter` to document the brand's visual rules into a shareable style guide.
