# Superpowers Core

Framework de développement logiciel structuré pour une productivité maximale.

## Skills actifs (Core)

1. **Writing Plans** : Création de plans d'exécution granulaires (tâches de 2-5 min).
2. **Test Driven Development (TDD)** : Cycle RED/GREEN/REFACTOR strict.
3. **Requesting Code Review** : Validation de la qualité avant le merge des tâches.
4. **Systematic Debugging** : Débogage structuré avec traçage de la cause racine.
5. **Verification before completion** : Checklist finale pour garantir la robustesse.

## Utilisation

Ces skills sont conçus pour être utilisés de manière fluide pendant le cycle de développement par l'agent.

## Stockage

Les composants avancés non utilisés immédiatement sont stockés dans `skills/superpowers-inactive/`.
